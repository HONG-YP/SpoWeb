
# 스포 적발 강력 1팀
## [스포의 스포] : 네 스포가 스포라는 것을 스포한다!

<pre>
빅데이터 분석 동아리 BOAZ 14기 ADV의 NLP 프로젝트입니다. <br> 
영화 리뷰에 대한 스포 여부 및 스포 확률을 조회할 수 있는 웹사이트입니다.<br>
특정 영화나 장르에 관계없이 범용적인 스포일러를 감지하는 것을 목표로 합니다. 
</pre>

## 필요 라이브러리
<pre>
transformers >= 4.2.1 <br>
keras >= 2.4.3 <br>
tf >= 2.4.0 
</pre>
